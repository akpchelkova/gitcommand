# Команды для GITа

## 1. Создает новую ветку

```sh
git branch - показывает существующие ветки
git branch <название ветки> - создает ветку
```

## 2. Переходит на другую ветку

```sh
git checkout <название ветки>
```

## 3. Удаляет ветку

```sh
branch -d 
```

## 4. Слияние с другой веткой

```sh
merge
```
